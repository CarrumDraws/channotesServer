Get > Post > Put > Delete

// LOGIN ----------
post("/auth/signup") : Make new user +  Returns {User}
get("/auth/hasuser") : Returns {User} from google_id. If none found, returns nothing.

// FOLDERS ---------
get("/folders") : Gets Nested Folders : Queries- folderid : If !folderid then return homepage folders
post("/folders") : Adds Folder
put("/folders") : Queries :  folder_id, title, parent_id
delete("/folders/:folderid) : Deletes Folder, Nested folders + notes, notephotos and shares

// NOTES ---------
get("/notes/") : Gets Notes : Queries- folder_id. If !folder_id then return homepage Notes.

// NOTE --------
get("/note/") : Gets Text of specific note
post("/note/") : Adds New Note
put("/note/") : Edit a Note’s Contents, Sets Time Accessed, Renames Note
put("/note/meta) : Queries: move, pin, password, lock, fontcolor, backgroundcolor
delete("/note/) : Deletes Note, notephotos and shares

// SHARED ----------
get(“/shared/:userid”) : Gets a User’s Shared Notes
get(“/shared/:userid/:noteid”) : Get Shared Notes between User and Friend
put(“/shared/:userid/:noteid”) : Toggles User’s Access to a Shared Note

// USERS ----------
get("/users/:userid") : Gets User Data
get("/users/:userid/friends") : Get Friends of User
get("/users/search") : Queries: name
post("/users/:userid") : Sets User Data
put("/users/:userid/:friendid") : Queries: friend, block
